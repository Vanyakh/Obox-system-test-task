Wakanda Це мій проєкт,який я зробив, в якості тестового завдання. 
Головна ідея була в тому, щоб знайти заміну складним 3д моделям які використовувались в оригунальному сайті Wakanda.

Що було реалызовано у цьому проєкті
Горизонтальний рух: замість звичного скролу вниз, для сворення ефекту панорами.

Ефект глибини: фон і заголовки трохи реагують на рух мишки.

Інтерактивна сцена: на другому етапі панорама зупиняється, текст відлітає вбік, і з'являються точки з інформацією. 

Плавність: я додав спеціальну бібліотеку для скролу, щоб рухи були м'якими та приємними для ока.

Які технології я використовував
Все зроблено на базі HTML, CSS та чистого JavaScript.

Для анімацій я обрав GSAP та ScrollTrigger. Також я використав Lenis для м'якого скролу.

Використання ШІ в ході виконання проекту
При розробці я звертався за допомогою до Gemini. 

ШІ допоміг мені швидше розібратися з логікою таймлайнів у GSAP.

Підказав, як краще розрахувати координати для точок.
